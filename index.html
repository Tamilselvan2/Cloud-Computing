Protocols.

How nodes share the wireless medium and how data finds its destination. Memorize S-MAC and protocol classifications.

Expected 15 Mark Question

Classification of MAC Protocols

MAC protocols dictate how nodes access the shared wireless channel. In WSNs, avoiding energy waste is the main goal.

Contention-Based (CSMA)

Nodes compete for the channel. If busy, they wait.

Scalable, adapts to changes
High energy waste from idle listening
Example: CSMA/CA

Schedule-Based (TDMA)

Time is divided into slots. Each node gets a slot.

Collision-free, eliminates idle listening
Requires strict clock synchronization
Nodes sleep when not their slot
★

S-MAC (Sensor-MAC)

Designed to reduce energy consumption from idle listening.

😴
Periodic Listen & Sleep

Nodes sleep, then wake up to listen

🔄
Synchronization

Virtual clusters with same schedules

📨
Message Passing

Long messages broken into fragments

Crucial for 7 Marks

Routing Protocols Classification

Routing in WSNs is challenging due to no global addressing and severe energy limits.

1. Proactive (Table-Driven)

Maintains routing tables constantly. High overhead, zero delay when sending.

Example: DSDV

2. Reactive (On-Demand)

Finds route only when data needs to be sent. Lower overhead, initial delay.

Example: AODV, DSR

3. Hybrid

Combines both approaches. Proactive for local, reactive for distant nodes.

Example: ZRP

4. Flooding / Power-Aware

Flooding broadcasts blindly. Power-aware routes through highest battery nodes.

Maximize network lifetime

Complex Environment

Underwater WSN (UWSN)

Oceanographic data collection, pollution monitoring, and offshore exploration.

🌊 The Major Challenge: Acoustic Channels

Radio waves attenuate rapidly in water. UWSNs use Acoustic signals (sound). Acoustic waves have high propagation delay (~1500m/s vs speed of light), low bandwidth, and suffer from high multipath fading and Doppler spread.

Link Estimation

Underwater channel changes rapidly (currents, temperature gradients). Static routing fails. Nodes must constantly measure link quality to neighbors using ETX.

Channel-Aware Routing

Routing protocol uses link estimates to dynamically select the most stable path, avoiding nodes with poor acoustic connections to prevent packet loss and save energy.
